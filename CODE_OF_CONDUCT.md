# Code of Conduct

This project aims to be a practical, respectful space for open source
maintainers and contributors.

## Expected Behavior

- Be kind and specific in feedback.
- Assume good intent while asking clear questions.
- Respect maintainer time and project scope.
- Credit ideas, examples, and prior work.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Posting private information without permission.
- Spam, brigading, or artificial engagement.
- Misrepresenting project metrics or contributor status.

## Reporting

Open a private security advisory or contact the maintainers through the
repository's listed maintainer channel. Reports will be reviewed with care.
